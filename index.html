Star Commander in RoK: Comprehensive Analysis of the Commander Leveling Formula

1. Introduction

In RoK, upgrading the "Star Level" of a commander is a crucial factor in determining the player's strength. To achieve this, players must use various types of "Star Materials" known as Star1, Star2, and Star3 (corresponding to bronze, silver, and gold). However, resources are limited, so optimizing the formula to maximize efficiency is a challenging task.

2. Issues and Objectives

  • Each time you upgrade, you can use up to 6 gems.
  • Each type of gem has a base EXP value and a bonus multiplier:
    • Star1: 800 EXP, +5% bonus per gem
    • Star2: 400 EXP, +20% bonus per piece
    • Star3: 100 EXP, +10% bonus per piece
  • The maximum bonus is 100%

Objective: Find the best upgrade formula for maximum EXP gain under resource-constrained conditions (especially Star1).

3. Formula calculation

For each formula using x units of Star1, y units of Star2, and z units of Star3:

stars = x + y + z
base_exp = 800 * x + 400 * y + 100 * z
bonus = min(1.0, 0.05 * x + 0.20 * y + 0.10 * z)

Explanation for expected_exp:

Initially, you have base EXP. You will receive base EXP x2 if spirit or base EXP

On average, you will have

expected_exp = base_exp * 2 * bonus + base_exp * (1 - bonus)

Thus, we have

expected_exp = base_exp * (bonus + 1)

4. Code to calculate and sort the results

results = []

# Loop through all possible combinations of Star1 (x), Star2 (y), and Star3 (z)
# such that the total number of stones (x + y + z) ≤ 6
for x in range(0, 7):
    for y in range(0, 7 - x):
        for z in range(0, 7 - x - y):
            stars = x + y + z
            base = 800 * x + 400 * y + 100 * z  # Base EXP for each stone type
            p = min(1.0, 0.05 * x + 0.20 * y + 0.10 * z)  # Bonus rate (capped at 100%)
            expected = base * (1 + p)  # Expected EXP including bonus
            results.append((x, y, z, stars, base, p, expected))

# Sort all combinations by expected EXP in descending order
results.sort(key=lambda x: x[6], reverse=True)

# Print top 20 best formulas
print("Top 20 Formulas:")
print("Rank | S1 | S2 | S3 | Stars | Base EXP | Bonus % | Expected EXP | Diff")
print("-" * 85)

prev_exp = results[0][6]  # Store expected EXP of top result to compare drop-off
for i, (x, y, z, stars, base, p, expected) in enumerate(results, 1):
    diff = prev_exp - expected
    print(f"{i:4d} | {x:3d} | {y:3d} | {z:3d} | {stars:5d} | {base:8d} | {p*100:6.1f}% | {expected:11.1f} | {diff:8.1f}")
    prev_exp = expected
